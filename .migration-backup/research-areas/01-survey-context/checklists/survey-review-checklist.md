# Survey Review Checklist

Use this checklist before starting analysis to ensure deep understanding of your survey.

---

## 1. Survey Design Understanding

- [ ] Identified sampling frame and method
- [ ] Documented stratification variables
- [ ] Located sampling weight variable
- [ ] Understood PSU (cluster) structure
- [ ] Reviewed response rates

## 2. Module Documentation

For each survey module:

### Module: _______________
- [ ] Written "story" explaining purpose
- [ ] Identified key research questions it answers
- [ ] Mapped to indicators it feeds
- [ ] Documented skip logic affecting this module

## 3. Question-Level Review

- [ ] Reviewed each question for construct measured
- [ ] Identified assumptions about respondent understanding
- [ ] Noted potential weaknesses/biases
- [ ] Flagged questions with recall period concerns
- [ ] Identified Likert items requiring reverse coding

## 4. Skip Logic Mapping

- [ ] Documented all `relevant` expressions
- [ ] Created list of structural NA patterns
- [ ] Distinguished structural from non-response missingness

## 5. Multi-Select Questions

- [ ] Listed all select_multiple questions
- [ ] Decided processing strategy for each
- [ ] Determined correct N denominator

## 6. Version Differences

- [ ] Compared Version A and B structures
- [ ] Listed columns unique to each version
- [ ] Documented "not collected" vs "missing" distinction

## 7. Vietnam Context Notes

- [ ] Researched typical diets in study area
- [ ] Documented seasonal considerations
- [ ] Reviewed food waste cultural attitudes
- [ ] Researched income patterns (formal/informal)
- [ ] Found relevant poverty line values
- [ ] Reviewed existing Vietnam food security literature

## 8. Construct Mapping

- [ ] Created construct-question mapping table
- [ ] Documented transformations needed
- [ ] Justified question selection for each construct

## 9. Limitations Documentation

- [ ] Listed interpretation guardrails
- [ ] Documented cross-sectional limitations
- [ ] Noted social desirability concerns
- [ ] Identified population exclusions

## 10. Reference Collection

- [ ] Gathered FAO/WFP methodology documents
- [ ] Found Vietnam-specific research papers
- [ ] Located relevant GSO/VHLSS statistics
- [ ] Bookmarked key package documentation

---

## Sign-Off

- [ ] All sections complete
- [ ] Context notes saved to `templates/vietnam-context-notes.md`
- [ ] Construct map saved to `templates/construct-mapping.md`
- [ ] Ready to proceed with data cleaning
